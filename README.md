# AniEdit-Tool (Python Version)

AniEdit-Tool is now a standalone Python program for downloading and editing anime episodes from AniWatch. The web interface has been removed.

## Features

- Download anime episodes from AniWatch
- Trim episodes by specifying start and end times
- Command-line interface (CLI)

## Usage

1. Install dependencies: `pip install -r requirements.txt`
2. Run the program: `python main.py`
3. Follow the prompts to enter the episode URL, output file, and time range

## Requirements

- Python 3.10+
- ffmpeg (must be installed and available in PATH)
